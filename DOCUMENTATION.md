# Документація сайту «1 СТО Дніпро TIR»

## Огляд

Сайт є статичним багатосторінковим проєктом на HTML5, CSS3 та чистому JavaScript ES modules. Збірка, npm-залежності та фреймворки не використовуються.

Основні канали звернення:

- телефон;
- Viber;
- Instagram Direct після додавання URL у конфігурацію.

## Сторінки

### `index.html`

Головна сторінка містить:

- topbar з адресою, графіком, телефоном, Viber та умовним Instagram;
- sticky header з логотипом, навігацією і кнопкою дзвінка;
- hero з основними CTA;
- сітку локальних SVG-файлів марок;
- три переваги;
- блок про СТО;
- основні послуги;
- блок виїзної технічної допомоги;
- розбірку DAF Euro 2;
- обслуговування автопарків;
- галерею;
- FAQ;
- фінальний контактний блок.

### `poslugy.html`

Сторінка послуг описує ремонт вантажних автомобілів, тягачів, причепів, напівпричепів, ходової, двигунів, ГУР, компресорів, зварювальні та слюсарні роботи. Контактний блок пропонує дзвінок, Viber та умовний Instagram.

### `vyizdna-tehdopomoga.html`

Сторінка виїзної технічної допомоги пояснює, які дані потрібно повідомити:

- місцезнаходження;
- марка та модель;
- тип транспорту;
- опис несправності;
- чи може автомобіль рухатися;
- фото або відео проблеми.

Можливість, час та умови виїзду погоджуються телефоном.

### `rozbirka-daf.html`

Сторінка розбірки DAF Euro 2 пояснює, як уточнити наявність запчастини. Потрібно повідомити модель DAF, рік випуску, VIN, номер деталі, назву або опис деталі та за можливості надіслати фото.

Сайт не показує вигадані товари, ціни або статуси наявності.

### `pro-nas.html`

Сторінка про СТО описує спеціалізацію, принципи роботи, підтверджені напрямки ремонту, роботу з компаніями та галерею.

### `kontakty.html`

Контактна сторінка містить:

- контактні картки телефону, Viber та умовного Instagram Direct;
- адресу;
- графік;
- основний і додатковий телефони;
- карту через Google Maps;
- кнопку маршруту;
- територію виїзної допомоги.

### `404.html`

Сторінка помилки з посиланням на головну та кнопкою дзвінка.

## Конфігурація

Файл `assets/js/config.js` містить змінні контактні дані:

```js
export const businessConfig = {
  name: "1 СТО Дніпро TIR",
  address: "вул. Верстова, 33, Дніпро",
  phones: [
    {
      label: "+380 63 409 89 03",
      href: "tel:+380634098903"
    },
    {
      label: "+380 97 821 34 58",
      href: "tel:+380978213458"
    }
  ],
  schedule: "Пн–Сб: 09:00–18:00",
  viberUrl: "viber://chat?number=%2B380634098903",
  instagramUrl: "",
  googleMapsUrl: ""
};
```

Якщо `instagramUrl` порожній, Instagram-кнопки та картки приховуються. Якщо `googleMapsUrl` порожній, сайт формує Google Maps search-посилання за адресою.

## JavaScript

### `assets/js/main.js`

Виконує:

- підстановку контактних даних із конфігурації;
- побудову Google Maps URL;
- підстановку Viber URL;
- умовне відображення Instagram;
- встановлення поточного року;
- запуск навігації, FAQ, галереї, reveal-анімацій і scroll-to-top.

### `assets/js/navigation.js`

Керує мобільним меню, sticky header, активним пунктом навігації та закриттям меню після кліку або Escape.

### `assets/js/gallery.js`

Створює lightbox для зображень із `data-lightbox-src`, підтримує кнопки попереднього/наступного фото, Escape і клавіші зі стрілками.

### `assets/js/faq.js`

Додає `aria-controls`, `aria-expanded` і поведінку accordion для FAQ.

## CSS

Стилі розділені на:

- `reset.css` - базове скидання стилів і focus-visible;
- `variables.css` - палітра, типографіка, тіні, радіуси;
- `base.css` - секції, контейнер, заголовки, списки, утиліти;
- `components.css` - кнопки, topbar, header, hero, brand grid, картки, галерея, FAQ, контакти, footer, mobile actions;
- `pages.css` - сторінкові композиції;
- `responsive.css` - адаптивність.

Палітра базується на білому, світло-сірому, темно-сірому та світло-блакитному.

## SVG марок

Папка `assets/images/brands/` містить placeholder-файли:

- `daf.svg`
- `man.svg`
- `volvo.svg`
- `iveco.svg`
- `scania.svg`

Це нейтральні локальні SVG, не офіційні копії. Їх потрібно замінити на надані клієнтом або офіційні brand assets, зберігши ті самі шляхи.

## SEO

У sitemap залишені тільки активні сторінки. На головній сторінці є JSON-LD для `AutoRepair`/`LocalBusiness` із телефоном, адресою, графіком, територією обслуговування та URL. FAQPage JSON-LD відповідає видимому FAQ.

У structured data не використовуються відгуки, AggregateRating або непідтверджені дані.

## Конфіденційність

Окрему сторінку політики прибрано, бо в поточній версії сайту немає аналітики, cookies або вбудованого збору персональних даних. Комунікація відбувається через зовнішні канали: телефон, Viber та Instagram Direct.
